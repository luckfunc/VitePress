### npm的命令
``` cmd
- npm init  # 初始化当前所有目录
- npm i  # 安装所有依赖
- npm i express # 安装目录默认到dependencies
- npm i express -g # 安装到全局目录
- npm i express -S # 安装到dependencies生产依赖
- npm i express -D # 安装到devDependencies开发依赖
- npm i less-loader@ 1.2.3 # 安装less-loader指定版本 1.2.3
```
